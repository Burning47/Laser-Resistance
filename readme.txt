README - Laser Resistance V 1.0.0 11/06/2014

 
Installation:

 In order to install Laser Resistance, insert the game disc into a disc drive. Run the setup.exe on the disc, and follow the instructions on the installer to finish the installation.

Uninstallation:
 
 To uninstall Laser Resistance, navigate to the folder of the installed program. There, run the unins000.exe.
